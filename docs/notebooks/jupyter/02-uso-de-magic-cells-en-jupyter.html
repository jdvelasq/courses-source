Uso de magic cells en Jupyter¶

  • 60 min | Última modificación: Junio 22, 2019.

Jupyter tienen varias utilidades que extienden las capacidades de los notebooks más allá de las características del kernel que se este usando. Las magics permiten modiicar el comportamiento de las celdas individuales. A continuación se describen las principales. Haga clic aquí para acceder a la documentación oficial.

Una lista de los magics disponibles se encuentra en: https://github.com/ipython/ipython/wiki/Extensions-Index

%%bash ejecuta el código de la celda usando bash.

[1]:

%%bash
echo 'hola mundo'

hola mundo

%%html ejecuta el código como html.

[1]:

%%html
<p id="demo">Texto para cambiar.</p>

Texto para cambiar.

%%javascript, %%js Ejecuta el codigo de la celda usando javascript.

[2]:

%%javascript
// cambia la salida de la celda anterior
document.getElementById("demo").innerHTML = "---> Nuevo texto <---"

%%markdown Ejecuta el código de la celda como markdown.

[9]:

%%markdown
# Titulo

## Subtitulo

Texto texto texto
* viñeta 1
* viñeta 2

Titulo¶

Subtitulo¶

Texto texto texto * viñeta 1 * viñeta 2

%%perl
use feature ':5.10';

%%python, %%python2, %%python3 Ejecuta la celda usando Python.

%%python
print('Hola mundo')

%%ruby
s = "Hi there.  How are you?"
print s.length, " [" + s + "]\n"
