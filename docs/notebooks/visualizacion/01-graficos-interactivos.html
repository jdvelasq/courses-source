 Bokeh¶

Bokeh es una librería para graficación desarrollada por Continuum Analytics. El tutorial de Bokeh puede consultarse en el sitio oficial de la herramienta, cuyo vínculo está suministrado en la bibiografía.

En este tutorial se presenta un gráfico en Bokeh que tiene como fin probar que la librería está correctamente instalada.

[7]:

# El primer paso consiste en importar las librerias en Jupyter.
import numpy as np
from bokeh.plotting import figure, output_file, show
from bokeh.io import output_notebook

[8]:

output_notebook()
Loading BokehJS ... 
[9]:

# se generan puntos aleatorios
N = 4000
x = np.random.random(size=N) * 100
y = np.random.random(size=N) * 100
radii = np.random.random(size=N) * 1.5
colors = [
    "#%02x%02x%02x" % (int(r), int(g), 150) for r, g in zip(50+2*x, 30+2*y)
]

[10]:

# genera la figura, grafica los puntos y visualiza la gráfica
p = figure(x_range=(0,100), y_range=(0,100))
p.circle(x,y, radius=radii, fill_color=colors, fill_alpha=0.6, line_color=None)
show(p)

Seaborn¶

Seaborn by Michael Waskom.

Seaborn es una librería para la visualización de datos estadísticos desarrollada en matplotlib por M. Waskom. El sitio web indicado en la bibliografía contiene información detallada incluyendo la guía de instalación, notas introductorias, un tutorial y el manual de referencia.

A continuación se realiza un gráfico (del tutorial) en Jupyter para verificar que seaborn funciona correctamente.

 MPLD3: Bringing Matplotlib to the Browser¶

Sitio web: http://mpld3.github.io

Instalación:

[ ]:

%%sh
conda install mpld3

[ ]:

%matplotlib inline
import numpy as np
import matplotlib.pyplot as plt
import mpld3
mpld3.enable_notebook()

[ ]:

from mpld3 import plugins
fig, ax = plt.subplots()
points = ax.scatter(np.random.rand(40), np.random.rand(40),
                    s=300, alpha=0.3)

labels = ["Point {0}".format(i) for i in range(40)]
tooltip = plugins.PointLabelTooltip(points, labels)

plugins.connect(fig, tooltip)
