Laboratorios de Programación
==========================================================================================

Para realizar los laboratorios de clase usted deberá realizar las siguientes actividades.

**Actividad 1**

  Realice la instalación de Vagrant y Docker siguiendo las instrucciones que aparecen en
  el menu vertical de la derecha.


**Actividad 2**

  Entre al vínculo https://classroom.github.com/xxx e identifíquese en la lista de 
  estudiantes del curso. El sistema creará automáticamente un repo para usted en 
  GitHub.

**Actividad 3**

  Clone dentro de la carpeta de la máquina virtual (vagrant4docker) el repositorio
  obtenido en la Actiivdad 2, usando GitHub Desktop. Esto permitirá que la carpeta
  con las tareas sea accesible desde dentro de la máquina virtual de Docker.

**Actividad 4**

  Realice el Taller Hola Mundo para verificar que las actividades pueden ser realizadas
  adecuadamente.
